Seamless Identity Management with Thinkific SSO Solution



Thinkific Single Sign-On (SSO) solution by miniOrange helps you simplify everyday teaching and learning experience for students, teachers, and staff by providing you a seamless Thinkific login experience. Users of Thinkific will no longer need to maintain separate credentials for their students or invest in any custom development to integrate with Thinkific. You can log into the Thinkific platform using your existing credentials. Thinkific Single Sign On (SSO) solution also makes life easy for both Administrators and end-users, as it provides the ability to securely exchange user identity data between Thinkific & end-user directories.

On top of this, the Thinkific Multi-Factor Authentication (MFA) solution adds an extra layer of protection for Thinkific by challenging users to a second factor. We provide 15+ authentication methods such as OTP over SMS, TOTP tokens, OTP over Email, hardware tokens, etc. to choose from.




Streamline Workforce and Customer security with designated features



Connect existing IdP's / User Stores

Use your existing IDP credentials for SSO Thinkific login. No need to onboard your users separately, we use existing user stores like IDP, Azure Active Directory (AAD)/ LDAP, etc.

Multiple IdP / User Store support

You can configure multiple IDPs/ User stores for Thinkific and give users the option to select the IDP of their choice to login. miniOrange provides a centralized way to connect seamlessly with all IDPs and integrate SSO into Thinkific.

IP / Device Restriction

Remember personal devices or restrict users to login from specific IPs providing additional security to the accounts.

Multiple Factor Authentication

Admin can enable or disable Thinkific as 2FA for a particular user and for any particular application. We offer to log in using backup codes whenever the user is not having the phone available.

User Sync

User Sync syncs users from any third-party platform to Thinkific and vice versa. It includes all features like update, delete, add, etc.

Thinkific as an OAuth provider

You can configure Thinkific as an Oauth provider to single sign-on into apps that support SAML or any other protocols like OAuth, OpenID Connect, JWT, etc.
